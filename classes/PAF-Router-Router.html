init()

Router.php : 67

Initializes the router by creating the root-group and setting if cors should be enabled.

public static init([string $basePath = '' ][, bool $corsEnabled = false ]) : void

Sets the cors-headers (if enabled) and all previousely set headers.

Tipp: If you want cors to work and you pass a authorization-header or a content-type header, you have to add a 'Access-Control-Allow-Headers' header with the wanted headers as value (separated by ',')

A initialization is necessary and can only happen once.

Parameters
$basePath : string = ''

The base path, from where the router should match
