public pagination(int $itemsPerPage, int $page) : PaginationResult

This function overwrites the set limit and offset properties.

The result has the following structure:
['page'] - int - The current page
['items_per_page'] - int - The amount of items contained by a page
['total_items'] - int - The total amount of items found
['total_pages'] - int - The total amount of pages needed to retrieve all items
['items'] - Collection/Array - A collection (not raw) / array (raw) containing the resulting items

Parameters
$itemsPerPage : int

The amount of items contained by a page

$page : int

The current page (starting with 0)
